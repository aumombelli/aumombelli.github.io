3rd year project - ISIMA

3rd year project of ISIMA engineering school.

Integration of learning methods in a metaheuristic to solve vehicle routing problems.

Meta-heuristics search the solution space by using local search operators, but a lot of useful information is lost during the execution of the algorithm. In this project we have insert a learning method in our meta-heuristic in hope to improve in precision and speed the program solving vehicule routing problems.

Modelisation and development of a simulator for the evolution and optimization of a client's assets.

The purpose of this internship was to create a assets robot prototype which, with the expectations of the client and his assets, is able to calculate the orders of financial investments to be made over the years so that the client's expectations are met and that the heritage is the best possible. 2nd year project - ISIMA

2nd year project of ISIMA engineering school.

ECM + Optimization Method: Approximating the variance-covariance matrix.

The aim of this project is to create a library of MatLab functions for teacher-researchers Violaine ANTOINE and Jonas KOKO for an automatic classification problem. It performs an approximation of the matrices corresponding to ellipses in n dimensions representing as closely as possible groups of data. Development was done with the Octave IDE on Windows 10 education. The tests were carried out in the MatLab IDE on Fedora/UNIX.
